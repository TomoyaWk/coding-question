# 問題4: ウィンドウ最大値

整数のリストとウィンドウ幅 `k` が与えられたとき、連続する長さ `k` の各ウィンドウの最大値を順に返す関数を作成してください。

## 要件
- `k` は 1 以上、`len(nums)` 以下とする
- 結果は左から右の順にリストで返す
- 効率を意識して、全ウィンドウを毎回走査する実装は避ける

## 関数シグネチャ
```python
def sliding_window_max(nums: list[int], k: int) -> list[int]:
    ...
```

## 例
入力:
```python
nums = [1, 3, -1, -3, 5, 3, 6, 7]
k = 3
```

出力:
```python
[3, 3, 5, 5, 6, 7]
```

## 追加例
入力:
```python
nums = [9, 8, 7, 6]
k = 2
```

出力:
```python
[9, 8, 7]
```
