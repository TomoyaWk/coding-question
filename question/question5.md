# 問題5: 括弧バランスチェッカー

文字列に含まれる括弧が正しく対応しているかを判定する関数を作成してください。

## 要件
- 対象となる括弧は `()`, `[]`, `{}` の3種類
- 開き括弧と閉じ括弧が正しい順序でペアになっている必要がある
- 括弧以外の文字は無視する
- 空文字列は `True` を返す
- 括弧の種類が混在していても、それぞれが正しく対応していれば `True`

## 関数シグネチャ
```python
def is_balanced(text: str) -> bool:
    ...
```

## 例
入力:
```python
text = "print(data[0])"
```

出力:
```python
True
```

## 追加例

入力:
```python
text = "{a: [1, 2, (3 + 4)]}"
```

出力:
```python
True
```

入力:
```python
text = "([)]"
```

出力:
```python
False  # 括弧の対応順序が不正
```

入力:
```python
text = "((("
```

出力:
```python
False  # 閉じ括弧が足りない
```
